* cleopatra.js - eXBuilder6 런타임 자바스크립트입니다.
* css/ - eXBuilder6 기본 테마파일들을 포함하는 디렉토리입니다.
* conf/default.js - 런타임 기본값 설정 파일입니다. cleopatra.js 보다 먼저 로드된 경우에만 효력이 발생합니다.
* cpr-adapters.js - eXBuilder6 내부에 임베디드 페이지를 통해 연결된 일반 HTML, JSP등 페이지에서 eXBuilder6와 연동을 위해 사용할 수 있는 통신 모듈 스크립트입니다.

- - -

* cleopatra.js - eXBuilder6 runtime JavaScript.
* css/ - This is the directory containing the eXBuilder6 default theme files.
* conf/default.js - Runtime default configuration file. It only takes effect if loaded before cleopatra.js.
* cpr-adapters.js - This is a communication module script that can be used for interworking with eXBuilder6 in general HTML and JSP pages connected through embedded pages inside eXBuilder6.

- - -

* cleopatra.js - eXBuilder6ランタイムJavaScriptです。
* css/ - eXBuilder6 基本テーマファイルを含むディレクトリです。
* conf/default.js - ランタイムのデフォルト設定ファイルです。 cleopatra.jsの前にロードされた場合にのみ有効になります。
* cpr-adapters.js - eXBuilder6の内部に埋め込みページを介して接続された一般的なHTML、JSPなどのページでeXBuilder6と連動するために使用できる通信モジュールスクリプトです。
